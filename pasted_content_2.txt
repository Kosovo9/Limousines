1. NORTH STAR METRIC

NORTH STAR: Monthly Booked Gross Merchandise Value (GMV)
- Definition: Total value of all rides booked (passenger fees + commission to drivers)
- Why: Indicates platform health and growth
- Target Month 1: $10k
- Target Month 3: $100k
- Target Month 6: $500k
- Target Month 12: $2.5M

5. FUNNEL ANALYSIS

Landing → Signup → First booking → Payment → Rating
- Landing: 1000 visitors
- Signup: 100 (10% conversion) ← If <10%, landing/copy issue
- First booking: 30 (30% of signups) ← If <30%, onboarding issue
- Payment: 25 (83% of bookings) ← If <80%, payment issue
- Rating: 24 (96% completed) ← If <90%, UX issue
